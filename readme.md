# Install jekyll

* Install Ruby
  https://github.com/rbenv/rbenv
* Install bundler
* Install Jekyll
  * Bundler installs Ruby gems based on the Gemfile
  * Run $ bundle install
* Run Jekyll via bundler:
  $ bundler exec jekyll serve

# Build site

$ bundle exec jekyll build
